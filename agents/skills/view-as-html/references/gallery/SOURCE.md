# Source & License

The 20 numbered HTML files and `index.html` in this folder are vendored from
Anthropic's reference repository
[`anthropics/html-effectiveness`](https://github.com/anthropics/html-effectiveness),
the companion to the blog post
[*The unreasonable effectiveness of HTML*](https://claude.com/blog/using-claude-code-the-unreasonable-effectiveness-of-html).

Released under the MIT License. They are bundled with this skill so the workflow
does not depend on a local clone; without them the skill would have to fall back
to memory, which is exactly the failure mode this skill exists to prevent.

**These are reference materials, not templates.** Read them to learn structure,
type roles, and the visual signature — then write the user's content fresh.
Do not copy literal sample data ("Acme", placeholder names, fictional figures)
into output.
